Finance Expenses Dashboard (Excel)

An interactive Excel dashboard designed to track and analyze personal or business expenses, spending habits, and savings patterns.
This project visualizes income, expenses, and category-wise spending to support smarter budgeting and financial planning.

Key Insights
1. Total Expenses: $36,795, primarily from Living and Discretionary categories
2. Top Expense Categories: Rent, Groceries, and clothes
3. Expense Breakdown: Sub-category insights including dining, medical, and charity
4. Income vs. Expense Trend: Visual comparison across months
5. Spending Patterns: View by account type, month, and category through interactive slicers
